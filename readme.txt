Clean your WordPress media library, detect unused files with smart scanning, and optimize storage with a fast, intuitive workflow.

== Description ==

Olivero Media Audit helps you identify what is really used in your Media Library and what can be safely cleaned.

The plugin includes an intelligent detection engine that scans core content references and media usage patterns to reduce false positives and help you recover disk space confidently.

**FREE features:**

* Smart scanning of media references in standard content.
* Fast media analysis dashboard for used vs unused files.
* Individual file cleanup workflow.

**PRO features:**

* Advanced detection for page builders like Elementor and Divi.
* Deep scan in metadata/options for complex references.
* Temporary Trash workflow with restore before permanent deletion.
* Enhanced bulk cleanup tools and advanced usage insights.

If you want to optimize WordPress storage, remove unused images, and keep your media library efficient, Olivero Media Audit provides a practical and reliable workflow.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/olivero-media-audit` directory.
2. Activate the plugin from the **Plugins** screen in WordPress.
3. Open **Olivero Media Audit** from the admin menu.
4. Start a new scan and review used/unused files.
5. Review and delete unused files as needed.

== Frequently Asked Questions ==

= Is it safe to use on production sites? =
Yes. The plugin is built to minimize false positives and provides a controlled cleanup flow.

= Does it support Elementor and Divi? =
Yes, advanced page builder detection is available in the PRO version.

= Is Temporary Trash included? =
Temporary Trash with restore flow is available in the PRO version.

= Can I use it without PRO? =
Yes. The FREE version includes core scanning and basic cleanup.

= How do I enable PRO features? =
Install both plugins: **Media Usage Checker (FREE)** and **Media Usage Checker PRO Addon**. Then activate your license from the License tab.
