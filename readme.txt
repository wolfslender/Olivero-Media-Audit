Clean your WordPress media library, detect unused files with smart scanning, and optimize storage with a fast, intuitive workflow.

== Description ==

OliveroDev Media Audit helps you identify what is really used in your Media Library and what can be safely cleaned.

The plugin includes an intelligent detection engine that scans core content references and media usage patterns to reduce false positives and help you recover disk space confidently.

**What is included (FREE):**

* Smart scanning of media references in posts, pages, and widgets.
* Media analysis dashboard showing used vs unused files.
* Estimated storage savings display.
* Cleanup workflow for deleting unused attachments.
* Batch scanning with configurable settings.
* Cron-based automatic scanning for ongoing maintenance.
* Works with images (JPEG, PNG, GIF, WebP, SVG).

**Need more? Upgrade to PRO for:**

* Deep Detection Engine (ACF, Divi, Elementor, WooCommerce)
* Storage Analysis with visual charts (MB/GB breakdown)
* Support for Documents (PDF, DOC), Videos, Audio, Archives
* Intelligent Trash System (move files before permanent delete)
* One-Click Instant Bulk Cleanup
* Professional UI with advanced animations

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/oliverodev-media-audit` directory.
2. Activate the plugin from the **Plugins** screen in WordPress.
3. Open **OliveroDev Media Audit** from the admin menu.
4. Start a new scan and review used/unused files.
5. Review and delete unused files as needed.

== Frequently Asked Questions ==

= Is it safe to use on production sites? =
Yes. The plugin is built to minimize false positives and provides a controlled cleanup flow.

= Where can I find it in wp-admin? =
Go to **Tools → OliveroDev Media Audit**.

= What file types are supported? =
Images (JPEG, PNG, GIF, WebP, SVG). PRO version includes support for documents, videos, and audio files.

= What scanners are supported? =
Posts, pages, custom posts, widgets, and theme options. PRO version includes ACF, Divi, and Elementor.
